# OpenGL Particle Sandbox

Интерактивная песочница для изучения современной графики на OpenGL.

Проект создаётся с целью глубокого изучения компьютерной графики, устройства графического конвейера и разработки систем визуализации на C++. В дальнейшем планируется развитие в полноценный движок визуальных эффектов и симуляции частиц.

## Статус проекта

🚧 **Проект находится в активной разработке.**
На данный момент репозиторий создаётся как учебно-исследовательский проект для изучения современной графики на OpenGL. Основной акцент сделан на проектировании архитектуры и постепенной реализации графических подсистем.
**Текущая версия не является демонстрационным проектом и не отражает конечный результат разработки.**
По мере реализации новых возможностей репозиторий будет регулярно обновляться.

## Планируемые возможности

* система частиц;
* GPU-рендеринг;
* управление камерой;
* работа с шейдерами;
* загрузка текстур;
* различные типы эмиттеров;
* физика частиц;
* освещение;
* постобработка изображения;
* визуализация дебажной информации.

## Используемые технологии

* C++;
* OpenGL;
* GLSL;
* GLFW;
* GLAD;
* CMake;
* Git.

## Планируемая архитектура

```text
OpenGLSandbox
│
├── Core
│
├── Renderer
│   ├── Shader
│   ├── Texture
│   ├── Mesh
│   └── Camera
│
├── ParticleSystem
│   ├── Emitters
│   ├── Forces
│   └── Simulation
│
├── Scene
│
└── Resources
```

Проект проектируется таким образом, чтобы основные подсистемы были независимы друг от друга и могли использоваться повторно.

## Сборка

### Требования

* Windows;
* Visual Studio 2022 или новее;
* CMake;
* OpenGL 4.x;
* GLFW;
* GLAD.

### Запуск

Клонируйте репозиторий:

```bash
git clone https://github.com/Weez9x/OpenGLParticleSandbox.git
```

Сгенерируйте проект с помощью CMake, выполните сборку и запустите приложение.
